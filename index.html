3D Image Carousel Animation

The 3D Image Carousel Animation is a visually engaging project that showcases a rotating 3D image gallery using only HTML and CSS. It utilizes CSS transformations, perspective, and animations to create a dynamic effect, enhanced by reflections. The project highlights pure CSS capabilities for creating interactive, JavaScript-free web components.

Home Automation Using Bluetooth

The Home Automation Using Bluetooth project enables remote control of household appliances via a Bluetooth-enabled microcontroller. It uses an Android app to send control signals, processed by the microcontroller to switch devices on/off. Implemented using C++ (Arduino) for embedded programming, it integrates real-time communication protocols for seamless and responsive automation.

Soft-Active Robotic Hand

The Soft-Active Robotic Hand utilizes Nitinol shape-memory alloy (SMA) actuators for flexible and responsive motion. The system is microcontroller-driven, with Arduino (C++) programming controlling actuation. Sensors provide real-time feedback, enabling adaptive grasping. The Finite Element Method (FEM) is used for structural analysis, ensuring optimized force distribution. The design integrates embedded control algorithms for smooth, precise movements.
